COMPETENCIA

COMPETENCIA

RESUELVE PROBLEMAS DE FORMA MOVIMIENTO Y LOCALIZACIÓN 

CAPACIDAD

Identificar y representar figuras geométricas:

Los estudiantes deben reconocer triángulos rectángulos, identificar sus elementos (hipotenusa, catetos) y representarlos de manera adecuada.

Realizar cálculos numéricos:

Aplicar operaciones aritméticas para resolver ecuaciones y encontrar valores desconocidos en problemas relacionados con el teorema.

Resolver problemas:

Utilizar el Teorema de Pitágoras como herramienta para resolver una variedad de problemas, tanto dentro como fuera del contexto matemático

DESEMPEÑO

Resolver problemas contextualizados que involucran el Teorema de Pitágoras, como calcular la altura de un árbol o la distancia entre dos puntos utilizando triangulación.
